# Customer Behavior Dashboard

## Project Objective

This project is made to understand how customers shop and what patterns can be found in their buying behavior. The goal is to turn raw data into simple insights that can help businesses make better decisions.


## Dataset Used

The dataset contains customer shopping details such as:

* Customer age and gender
* Product category
* Purchase amount
* Discount applied
* Shipping type
* Ratings and reviews
* Order and payment information

- <a href = "https://github.com/reeti-verma/Customer-Behavior-Dashboard/blob/main/Dataset/customer_shopping_behavior.csv">Dataset Link</a>

## Questions (KPIs)

In this project, I tried to answer these questions:

* Which products are sold the most?
* How much total revenue is generated?
* Which customer group spends more?
* Which shipping method is most used?
* How do age and gender affect purchases?
* What is the average customer rating?
* Do discounts increase sales?


## Process

* Collected and understood the dataset
* Cleaned the data (removed missing values and errors)
* Performed basic analysis using SQL / Python
* Created important KPIs
* Built visual charts in a dashboard
* Found patterns and insights from the data


## Dashboard

An interactive dashboard was created to show:

* Sales performance
* Product category analysis
* Customer behavior
* Shipping trends
* Revenue insights

## Project Insights

* A small number of customers contribute most of the sales
* Some product categories perform better than others
* Most customers prefer standard shipping
* Discounts have a clear impact on buying behavior
* Customer ratings are mostly positive
